This assistant allows you to visually build different weekly schedules, check for time conflicts, and choose the best combination of professors for your semester.

## âœ¨ Key Features

* **Interactive Schedule Grid:** View the weekly schedule in a GUI (Saturday to Wednesday, 7 AM to 6 PM).

* **Dark Mode Support:** Smooth theme toggle button for eye comfort.
* **Auto-Save (Local Storage):** All plans and modifications are automatically saved in the browser and persist across page reloads.
